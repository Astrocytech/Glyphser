# Security Policy

## Security Contact
- Email: `security@astrocytech.com`
- PGP (optional): include your public key fingerprint in release metadata.

## Disclosure Policy
- Report suspected vulnerabilities privately to the security contact.
- Include reproducible steps, affected version/commit, and impact.
- Do not open a public issue before acknowledgment.
- Initial acknowledgment target: 3 business days.
- Status update target: every 7 calendar days until resolution.

## Supported Scope
- In scope: code and release artifacts in this repository.
- Out of scope: third-party services and local environment misconfiguration outside repository controls.
