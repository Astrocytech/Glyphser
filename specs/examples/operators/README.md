# Operator Examples (Illustrative)

Files in this folder are illustrative examples for specification readability.

## Boundary Rule
- These files are **not** executable conformance vectors.
- Canonical executable operator vectors live under:
  - `artifacts/inputs/conformance/primitives/operators/`
- Tests and tooling must load vectors from the canonical artifacts location.

## Curated Scope
- This folder intentionally keeps only a minimal illustrative subset:
  - `Glyphser_Model_Forward.json`
  - `Glyphser_IO_SaveCheckpoint.json`
- Exhaustive operator coverage belongs only to canonical conformance vectors in artifacts.
