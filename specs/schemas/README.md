# Schemas

Machine-readable schemas aligned with the layered spec model.

## Contents
- `specs/schemas/layers/L1/`: foundational schemas.
- `specs/schemas/layers/L2/`: runtime/system schemas.
- `specs/schemas/layers/L3/`: test/conformance schemas.
- `specs/schemas/layers/L4/`: implementation/ops/tooling schemas.

## Governance
Schema-to-spec mapping is enforced by:
- `governance/structure/spec_schema_map.json`
- `tooling/quality_gates/spec_schema_map_gate.py`
