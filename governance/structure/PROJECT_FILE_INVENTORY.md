# Project File Inventory Policy

The generated project inventory is evidence, not governance source-of-truth.

- Generator command: `python3 tooling/validation/generate_project_inventory.py`
- Generated output: `evidence/structure/PROJECT_FILE_INVENTORY.md`
- Template contract: `governance/structure/templates/PROJECT_FILE_INVENTORY_TEMPLATE.md`
- Pipeline integration: `tooling/push_button.py`
