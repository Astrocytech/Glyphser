# Governance

Policy and control plane for structure, roadmap, lint, and ecosystem metadata.

## Contents
- `governance/roadmap/`: milestone and phase planning.
- `governance/structure/`: structural policies, templates, and invariants.
- `governance/ecosystem/`: ecosystem graph/registry/validation logs.
- `governance/lint/`: semantic lint reports.
- `governance/process/`: contribution/review/governance process docs.
