# Annual Independent Security Review Checklist

- [ ] Confirm reviewer independence (reviewer is not a core maintainer for reviewed controls).
- [ ] Confirm review period and scope (authz/audit, supply chain, incident response, evidence integrity).
- [ ] Validate representative signed evidence artifacts and verification outputs.
- [ ] Document findings with severity, owner, remediation due date, and closure evidence path.
- [ ] Confirm unresolved high-severity findings have explicit risk acceptance or blocking action.
- [ ] Produce signed annual independent review attestation artifact.
